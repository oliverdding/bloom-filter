# bloom-filter

[![github-actions-badge][]][github-actions] [![release-badge][]][cargo] [![docs-badge][]][docs] [![rust-version-badge][]][rust-version]

Bloom filter implementation in rust with roaring-bitmap.

Now contains:

1. StableBloomFilter
2. VariantBloomFilter
3. ScalableBloomFilter

## Installation

```toml
[dependencies]
```
